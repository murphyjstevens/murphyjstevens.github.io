Projects

Jot It Down

Node JS

Jot It Down is an app that lets you take and save notes so that you can access them later. I created it using Node JS and relying on socket io to get the data back and forth between server and client. Since I'm still new to javascript/jquery and Node JS, I'd say the more difficult part of this project was getting information back when you select the note in the client.

While this project is fully functional, there are still some things that could be added, but I wanted to move on to other projects. If I were to put more work into it, I would add features such as email verification, ssl certifcation, a complete UI redesign, and real-time automatic saving.

Easy Chat

Node JS

EasyChat is a simple and clean chatroom web app that I created using Node JS. It started out as a project made through a Udemy course, but I added quite a few new features including unique names, random colors for users, and case-insensitive rooms.

If I were to add more to it later, I would add the ability to log in as a "verified user" which would give you a little icon by your name, load in existing chatrooms in a dropdown so you could either put in your own or choose from that list, and move away from the simple http post to make it look more clean.
